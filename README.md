# 📝 NotesApp Pro - تطبيق ملاحظات احترافي

[![Kotlin](https://img.shields.io/badge/Kotlin-1.9.0-purple.svg)](https://kotlinlang.org)

تطبيق ملاحظات متطور ومتكامل بنظام Android، يتميز بواجهة مستخدم عصرية وميزات متقدمة.

## ✨ الميزات الرئيسية

- ✅ **Material Design 3** - تصميم عصري ومتجاوب
- ✅ **وضع الليل/النهار** - تلقائي ويدوي
- ✅ **قفل التطبيق** - بصمة/نمط/رقم سري
- ✅ **ملاحظات صوتية** - تسجيل وتشغيل
- ✅ **إرفاق ملفات** - PDF, صور, فيديو
- ✅ **تذكيرات ذكية** - زمنية وجغرافية
- ✅ **مزامنة سحابية** - Firebase + Google Drive
- ✅ **إحصائيات متقدمة** - تحليل الإنتاجية
- ✅ **مشاركة متعددة** - QR Code, NFC
- ✅ **تحكم صوتي** - أوامر صوتية
- ✅ **ذكاء اصطناعي** - اقتراح الكلمات

## 📱 متطلبات التشغيل

- **الحد الأدنى:** Android 7.0 (API 24)
- **الهدف:** Android 14 (API 34)
- **الذاكرة:** 4GB RAM
- **المساحة:** 50MB
